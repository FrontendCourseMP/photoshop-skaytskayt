# GrayBit Image Editor — лаб. 2

Каналы и пипетка поверх редактора из лаб. 1.

- **Панель каналов** — миниатюры в градациях серого (1/2/3/4 канала по источнику),
  чекбоксы переключают видимость канала на главном холсте. Оригинальные пиксели
  не модифицируются — для отображения собирается новый массив.
- **Пипетка** — кнопка на тулбаре. По клику на холсте выводит координаты (X, Y),
  RGB (0..255), Alpha и CIELAB (D65).
